# What's Happening in This Lab

## What Was Set Up For You

When this lab started, the following happened automatically in the background:

- **Ubuntu 22.04** VM was provisioned
- **Python 3** was installed along with `pip`, `venv`, and `nano`
- A **Python virtual environment** was created at `/root/.venv`
- **Flask** and **requests** libraries were installed inside the virtual environment
- A **Flask web application** (`app.py`) was created at `/root/app.py`
- The Flask app was started automatically and is now **running on port 5000**

---

## What the Flask App Does

The app is a simple **Year Converter** web application that:

- Serves an HTML page at `http://localhost:5000`
- Exposes an API endpoint at `http://localhost:5000/api/server-info` that returns the server's private and public IP addresses
